# 🏥 AI Healthcare Chatbot

An AI-powered healthcare assistant that provides:
- 🤖 Chat-based medical guidance
- 🎤 Voice input (speech-to-text)
- 📍 Hospital locator
- 📊 Health dashboard

---

## 🚀 Tech Stack
- Frontend: React / Next.js
- Backend: Node.js
- AI: Google Generative AI
- Cloud: Microsoft Azure
- Database: Azure SQL

---

frontend/ → UI  
backend/ → APIs
